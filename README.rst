This repository demonstrates how respyabc can be used in an exemplary workflow of a research project using pytask. There are no tests written in this file here since all functions are already tested within respyabc.

For more information on the package check out respyabc at its `docs <https://respyabc.readthedocs.io/en/latest/>`_ or its GitHub `repository <https://github.com/manuhuth/respyabc>`_.
